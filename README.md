- **Версия Python**: 3.13.7
  - **Создать виртуальное окружение**: создайте виртуальное окружение с помощью команды  `py -m venv venv`.
  - **Активируйте виртуальное окружение**: активируйте виртуальное окружение командой `venv\scripts\activate`.
  - **Установка зависимостей**: выполните команду `pip install -r requirements.txt`.
  - **Команды запуска**: запустите приложение командой `streamlit run src\app.py`.
  - **Краткое описание структуры проекта**: проект содержит основные файлы `app.py`, модули `data_processing.py` и `visualization.py`, а также файлы настроек.
